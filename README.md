# Kehinde-O.O
Hi, I am Kehinde Olakunle Olawole, a Data Analyst with experience in SQL, Python and Excel. I specialize in transforming raw data into actionable business insights and interactive dashboards
Project Overview: This project is an interactive sales analytics dashbaord built in Microsoft Excel to analyze the business performance of Meva Coffee performance. It transforms raw sales data into meaningful insights to support data driven-making across sales,customers,products and operations. This dashboard helps stakeholders quickly understand performance trends, identify key drivers of revenue, and evaluate operational efficiency.
Objectives:  Analyze overall sales performance. Identify top performing product category and type. Evaluate the peak transaction period and monthly sales revenue.
Dashboard features: Sales trend over time. Top 10 products by sales. Peak Periods transactions.Location revenue performance
Key Insights: Revenue Performance:
